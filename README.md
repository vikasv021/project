<h1>Face Detection Web Application</h1>
<br>
Description:
This project is a web-based image recognition system that detects and counts human faces in uploaded images.

Technologies Used:
- Python
- OpenCV
- Flask
- HTML/CSS

How It Works:
1. User uploads an image.
2. The system processes the image using OpenCV.
3. Faces are detected using Haar Cascade classifier.
4. The result is displayed in the browser with detected face count.
